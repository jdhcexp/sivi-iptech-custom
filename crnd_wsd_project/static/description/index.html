Website Service Desk (Project)


This module provides integration between the Website Service Desk and Generic Request Project addons. It allows to see tasks related to the current request from the website interface.

On the request form, users will be able to see related projects and tasks.


Users can open project or task to see all required information.


This module is part of the Bureaucrat ITSM project. You can try it by the links below.
